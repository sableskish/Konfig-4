# Konfig-4
## Общее описание
Разработан инструмент командной строки для преобразования конфигурационных файлов, написанных на учебном конфигурационном языке, в стандартный формат JSON. Инструмент представляет собой парсер, который считывает текст из входного файла, анализирует его синтаксис, проверяет на наличие ошибок и генерирует эквивалентное JSON-представление данных. Учебный язык поддерживает различные типы данных, константы, комментарии и вложенные структуры.

## Описание всех функций и настроек
Инструмент состоит из основного класса ConfigParser, который содержит методы для обработки всех конструкций учебного языка. Основные функции включают:

Удаление комментариев: поддерживаются однострочные комментарии (начинающиеся с ||) и многострочные комментарии (заключенные между (* и *))

Парсинг чисел: целые числа и числа с плавающей точкой (формат \d*.\d+)

Обработка строк: строки заключаются в двойные квадратные скобки [[...]]

Работа с массивами: массивы определяются фигурными скобками с элементами, разделенными точками { элемент. элемент. ... }

Управление константами: объявление констант через оператор := и их использование через !(имя_константы)

Парсинг пар ключ-значение: в основном формате используется синтаксис ключ = значение

Обработка вложенных структур: поддержка произвольной вложенности массивов и объектов

Валидация имен: имена переменных и констант должны соответствовать шаблону [a-zA-Z][a-zA-Z0-9]*

Настройки инструмента задаются через аргументы командной строки:

Обязательный аргумент -i или --input: путь к входному файлу с конфигурацией на учебном языке

Обязательный аргумент -o или --output: путь к выходному файлу в формате JSON

При возникновении синтаксических ошибок инструмент выводит сообщения об ошибках в стандартный поток ошибок и завершает работу с ненулевым кодом возврата

## Описание команд для сборки проекта и запуска тестов
Запуск инструмента: python config_parser.py -i путь_к_входному_файлу.conf -o путь_к_выходному_файлу.json

Запуск тестов: python -m pytest test_config_parser.py -v
